I moved my blog to GitHub. It is best suited to share and work together on code any programming language (R in my case).  
_Mudé mi blog a GitHub. Es más adecuado para compartir y trabajar en conjunto en códigos de cualquier lenguaje de programación (R en mi caso)._   

* Using Markdown facilitates publish posts with code. Each post can also be associated with a project.  
* I'll begin with reproducible research, making available analysis routines and the original data.  
* Beyond the analysis codes, I can host on GitHub development packages (through integration with RStudio) and code for educational purposes.  
----
* El uso de Markdown facilita publicar posts que contengan código. Cada post puede además asociarse a un proyecto.  
* Comenzaré con trabajos de investigación reproducibles, poniendo a disposición las rutinas de análisis y los datos originales.  
* Más allá de los códigos de análisis de datos, en GitHub puedo albergar paquetes en desarrollo (mediante su integración con RStudio) y código con fines educativos.
